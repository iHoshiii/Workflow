1. High-Level Design (HLD)
This focuses on the "Big Picture." You decide how the system will be structured as a whole.
System Architecture: Choosing between styles like Monolithic, Microservices, or Client-Server.
Module Definition: Identifying the main parts of the app (e.g., User Auth, Task Engine, Notification Service).
Technology Stack: Finalizing the languages and tools (e.g., Python, PostgreSQL, React).

2. Low-Level Design (LLD)
This is the "Granular Detail." It focuses on the internal logic of each module.
Database Schema: Designing table structures, keys, and relationships.
Class Diagrams: Defining the objects, their attributes, and how they interact.
API Specs: Detailing the inputs and outputs for every function or endpoint.
